## 📊 איך לנתח את התוצאות?

### **ספירה דינמית ב-Google Sheets:**
```
=COUNTIF(Sheet2!C:C,"כן")  // כמה אנשים אמרו כן למרפסת
=COUNTIF(Sheet2!D:D,"כן")  // כמה אנשים אמרו כן לממ״ד
```

### **חתכים לפי בניין:**
```
=COUNTIFS(Sheet2!A:A,"טאגור 31",Sheet2!C:C,"כן")
=COUNTIFS(Sheet2!A:A,"טאגור 33",Sheet2!D:D,"כן")
```
